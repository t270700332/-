 ## 系统工作原理：
   ### 前置工作：
      此软件系统主要是为项目、合同的创建、开票以及各种费用的报销所用。各种功能涉及的最关键的部分就是审批流程。审批流程是自定义的，
      可以创建流程时选择所需要的功能。
 ![流程图片](img/1573735471(1).png)
      在报销审批的流程中要使用表单，所以我们也要先创建好表单，在报销、项目、借款、合同的提交中还要使用到编号，所以编号也要先创建好。
